### About

This is my repository for **my Rook MK2**, a 3D printer released in 2026 by [Rolohaun](https://www.printables.com/@Rolohaun), and it contains the files I have collected to build and operate my 3D printer.

> **This repository is not meant to be an official repository or public source of information for the official Rook MK2 project. Any information contained in this repository is copyright its respective license.**
>
> ### Official information on the Rook MK2: [![Discord](https://img.shields.io/badge/Discord-Rolohaun-7289da?logo=discord&logoColor=white)](https://discord.gg/GtSchMK44J) [![Printables](https://img.shields.io/badge/Printables-Model-orange.svg)](https://www.printables.com/model/1644597-rook-mk2)

This repository is specific to my printer vs. the official printer. I have made specific design choices for **my printer** that I would like to remember and therefore this repository was created. All the files contained herein were sourced publicly from Printables, Discord, the Voron github or my own creation.

* If you find this useful or have any suggestions please feel free to open an issue or pull request.

Thanks for visiting!

***John***
